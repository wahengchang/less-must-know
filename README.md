#less-must-know

# Install
```
$ npm install
```

# Build CSS from LESS
```
$ ./node_modules/less/bin/lessc style.less style.css
```

# Join less file (gulp)
Three module is needed: *gulp, gulp-less and gulp-concat*
```
./node_modules/gulp/bin/gulp.js buildCss
```
